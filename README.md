# TASK-7
## Auditing browser extensions
This task focuses on identifying, analyzing, and removing potentially harmful or unnecessary browser extensions. Browser extensions can improve productivity, but malicious or poorly designed extensions may compromise security, privacy, and performance

## objective

Audit installed browser extensions

Analyze permissions and reviews

Identify suspicious or unused extensions

Remove unsafe extensions

## Tools Used

Google Chrome / Mozilla Firefox

Browser Extension Manager

Online reviews (Chrome Web Store / Firefox Add-ons)

## Steps

Open Extension Manager

Review Installed Extensions

Check Permissions

Verify Reviews & Publisher

Identify Suspicious Extensions

Remove Unnecessary Extensions

Restart Browser
