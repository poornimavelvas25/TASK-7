# STEPS OF AUDITING

## Open Extension Manager
For Google Chrome:

chrome://extensions/

For Mozilla Firefox:

about:addons

## Review Installed Extensions

Checked all installed extensions

Noted extension names and purposes

Identified extensions no longer in use

## Check Permissions

For each extension, reviewed:

Access to all websites

Access to browsing history

Read and change data on websites

Background activity

Red Flag Permissions:

“Read and change all your data on all websites”

“Access browsing history”

“Manage downloads”

“Communicate with cooperating websites"

## Verify Reviews & Publisher

Checked ratings on extension store

Read user feedback

Verified developer legitimacy

Checked number of downloads

Red flags:

Very low ratings

Few downloads

Suspicious reviews

Unknown developer

## Identify Suspicious Extensions

Not used in last 30 days

Excessive permissions

Poor ratings

Unknown source

## Remove Unnecessary Extensions

Removed unused extensions

Removed extensions with risky permissions

Kept only trusted and necessary tools

## Restart Browser

Restarted browser

Observed performance improvements

Noted reduced RAM usage

Faster page loading time

# How Malicious Extensions Harm Users

Malicious browser extensions can:

Steal login credentials

Track browsing activity

Inject malicious ads

Redirect search results

Install malware

Collect sensitive personal information

## Conclusion

This project successfully demonstrated how a simple browser extension audit can significantly improve both security and performance.
By carefully reviewing installed extensions, analyzing permissions, verifying developer credibility, and removing unnecessary or suspicious add-ons, the browser environment became safer and more efficient.
